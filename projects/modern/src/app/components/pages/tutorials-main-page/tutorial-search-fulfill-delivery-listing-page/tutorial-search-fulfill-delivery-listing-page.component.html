Search for and Fulfilling a Delivery Listing (Service Provider)

As a service provider using Stripe Connect, you are able to claim delivery listings on the app.

While logged in, tap the "Search" button on the bottom navigation.

The initial screen will be a map. There are 3 buttons at the top right: the list icon is for searching for listings via selected pickup and dropoff locations, the location arrow icon is for getting your current location to pan the map to your position. The refresh/recycle arrow icon is for refreshing results based on the current bounds of the map (the areas visible/in view on map). It is recommended to use the map search instead of the list search.

After touching the refresh icon, If there are NO results, an alert will show to indicate that.

After touching the refresh icon, If there ARE results, they will be represented as markers on the map. Tap the marker to open the delivery listing preview.

The delivery preview will give a brief summary of the listing: the pickup and dropoff locations, date created, the payout, the distance between, etc. If you accept the payout, tap the "Claim" button to start the fulfillment promise.

The delivery is now assigned to you. As a service provider, you can see your active delivering by tapping the "Delivering" tab on the bottom navigation.

Tapping a list item will take you to the delivery listing details screen for that delivery. Swiping down from the top will refresh the screen with the latest information available.

Similar to the customer perspective, you will see details about the delivery listing. As the carrier, you have more controls/actions on the delivery listing, such as providing tracking updates, sending messages, capturing IDs/Signatures, marking statuses, etc.

From the delivery details screen, the controls for the service provider are located towards the bottom. From there, the app and UI will guide on how to use them.

After marking the delivery as dropped off, the work of the service provider is done. All that is left is to wait for the listing owner to review and payout. At that point, the delivery will be marked as complete.
